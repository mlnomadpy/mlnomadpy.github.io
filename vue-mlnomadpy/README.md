# MLNomadPy Vue.js Portfolio

This is a Vue.js rewrite of the MLNomadPy personal portfolio website. The site showcases the professional journey, research, and contributions of Taha Bouhsine, an ML researcher and Google Developer Expert in AI/ML.

## Features

- Modern Vue 3 implementation with Vue Router
- Responsive design that works on all device sizes
- Interactive components with smooth animations
- Sections for Research, Talks, Poetry, and more
- Data-driven content using JSON files

## Project Structure

- `src/components/` - Reusable Vue components
- `src/views/` - Page-level Vue components
- `src/assets/` - CSS, images, and other static assets
- `src/data/` - JSON data files for research, experiences, etc.
- `src/router/` - Vue Router configuration

## Setup and Development

### Prerequisites

- Node.js and npm

### Installation

```bash
# Navigate to the project directory
cd vue-mlnomadpy

# Install dependencies
npm install

# Start development server
npm run serve
```

### Building for Production

```bash
npm run build
```
