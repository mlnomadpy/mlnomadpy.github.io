ML In Depth: 1 - Deep Computer Vision - Classification

The ML workshop on Deep Computer Vision - Classification covered the basics of using Deep Learning for image classification. Participants learned about Convolutional Neural Networks (CNNs), and how to train their own models using TensorFlow. The workshop included a practical exercise where attendees built a web application to interact with their trained models in real-time. They also learned about evaluation metrics and strategies to improve model performance. No prior experience in computer vision or Deep Learning was needed, making the workshop accessible to everyone. Overall, it provided valuable knowledge and practical skills for exploring image classification with Deep Learning techniques.
