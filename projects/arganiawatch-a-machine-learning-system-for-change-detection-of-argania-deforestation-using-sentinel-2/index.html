ArganiaWatch - A Machine Learning System for Change Detection of Argania Deforestation using Sentinel-2

Overview:

ArganWatch is an innovative project that focuses on developing a cutting-edge Machine Learning System for change detection of Argania deforestation using Sentinel-2 satellite imagery. Leveraging the powerful ArganiaDetector model, previously developed for Argania tree detection, we feed it geolocation data and extract Sentinel-2 time series values from 2015 to 2022. By analyzing the temporal changes in the pixel values, the system accurately determines whether the Argania trees have been affected by deforestation or are still intact.

System Overview:

The foundation of the ArganWatch project is the ArganiaDetector model, which has been trained on Sentinel-2 pixel values to identify Argania trees with a high accuracy rate of 90%. To repurpose this model for change detection, we feed it specific geolocation data, and it analyzes Sentinel-2 time series values covering the period from 2015 to 2022. The system then detects changes in the Argania tree cover, helping identify potential deforestation areas.

Machine Learning Approach:

The ArganWatch system utilizes the TensorFlow Decision Forest library, which powered the original ArganiaDetector model, to perform change detection. The model processes the Sentinel-2 pixel values extracted from the time series data and predicts whether Argania trees have been deforested or are still present. By capitalizing on the pre-trained model, we efficiently repurpose it for a new application, saving computational resources and time.

Application Flow:

The ArganWatch application follows a structured flow to detect changes in Argania tree cover:

  1. User Input: Users provide geolocation data through the application interface.

  2. Time Series Data Extraction: The system retrieves Sentinel-2 imagery time series data for the specified location from 2015 to 2022.

  3. Feature Extraction: The ArganiaDetector model processes the time series data, extracting relevant features for change detection.

  4. Change Detection: Using the trained model, the system analyzes the extracted features and predicts whether Argania trees have been deforested or are still existing.

  5. Results and Reporting: The application provides users with detailed reports and visualizations, highlighting the detected changes and potential deforestation areas.

Conclusion:

The ArganWatch project introduces a powerful Machine Learning System for change detection of Argania deforestation using Sentinel-2 satellite imagery. By repurposing the ArganiaDetector model, we efficiently analyze temporal changes in pixel values to detect potential deforestation areas from 2015 to 2022. The system's accurate change detection capabilities offer valuable insights to researchers, environmentalists, and policymakers, facilitating timely interventions for preserving and conserving the Argania tree population. ArganWatch emphasizes the importance of monitoring and protecting Argania trees, contributing to sustainable land management and promoting a greener and healthier environment for future generations.
