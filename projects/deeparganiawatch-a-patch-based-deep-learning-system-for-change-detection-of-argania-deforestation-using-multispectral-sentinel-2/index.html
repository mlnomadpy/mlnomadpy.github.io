DeepArganiaWatch - A Patch-Based Deep Learning System for Change Detection of Argania Deforestation using Multispectral Sentinel-2

Overview:

DeepArganiaWatch is an advanced project that focuses on developing a Patch-Based Deep Learning System for change detection of Argania deforestation using Sentinel-2 multispectral imagery. The project comprises two major components: DeepArganiaDetector, a deep learning segmentation model specifically designed for Argania tree detection, and a change detection module that utilizes DeepArganiaDetector to detect and visualize deforestation changes between two dates specified by the user.

DeepArganiaDetector:

The foundation of DeepArganiaWatch lies in the development of DeepArganiaDetector, a deep learning segmentation model. DeepArganiaDetector is trained to take 64x64 pixel patches from Sentinel-2 multispectral imagery, consisting of 13 channels. It performs semantic segmentation to isolate Argania trees by outputting a 64x64 binary mask where "1" denotes the presence of Argania trees and "0" represents the background.

Training DeepArganiaDetector:

DeepArganiaDetector is pre-trained on the EuroSAT dataset using a U-Net architecture, producing an output of 64x64x13 for Argania tree segmentation. The last layer of the model is then modified to output a 64x64x1 binary mask to perform accurate Argania tree segmentation on our custom dataset. The model is fine-tuned on our dataset to achieve precise segmentation of Argania trees.

Change Detection Module:

The Change Detection module is an essential part of DeepArganiaWatch. Users input two dates and a 64x64 region of interest. DeepArganiaDetector is applied separately to the Sentinel-2 imagery from each date, producing two binary masks representing Argania tree presence for each date. The change map is then generated by analyzing the differences between the two binary masks. This process allows us to visualize and quantify deforestation changes in the specified region over time.

DeepArganiaWatch - Advancing ArganiaWatch:

DeepArganiaWatch is an improved version of ArganiaWatch, enhancing the accuracy and performance of change detection through the utilization of the DeepArganiaDetector model. By employing deep learning techniques and multispectral Sentinel-2 imagery, DeepArganiaWatch provides a more precise and comprehensive analysis of Argania deforestation, empowering researchers and environmentalists to make informed decisions for conservation and sustainable land management.

Conclusion:

DeepArganiaWatch represents a significant advancement in change detection of Argania deforestation using multispectral Sentinel-2 imagery. The integration of DeepArganiaDetector, a powerful deep learning segmentation model, allows us to precisely identify Argania trees and visualize changes in the tree cover over time. This project's impact extends to environmental conservation efforts, providing valuable insights to policymakers and researchers in preserving and safeguarding the Argania tree population. DeepArganiaWatch builds upon the foundation of ArganiaWatch, enabling more accurate detection of deforestation changes and enhancing our understanding of the dynamic landscape of the Argania trees in the Souss region.
